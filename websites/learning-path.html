From Zero to Hired

A complete, self-study path into software engineering. Eight courses, built to be worked in order, each ending in a real project for your portfolio.

Free & open source — forever

This is a learning path, not a random pile of courses. Each one assumes what came before it and sets up what comes next. Worked in order, they take you from "I've never written a line of code" to "I can build real software and talk about it like an engineer."

Every course ends with a capstone project — something you build, deploy, and put on your GitHub. By the end you don't just know things; you have a portfolio that shows you know them. That's what gets interviews.

Take your time. Do the exercises. Build the projects for real — don't just read the solutions. The people who finish this path and build every project will be genuinely employable.

Stage 1 · Foundations
01Programming Foundations
18 sections · Python

The absolute beginning. Variables, logic, loops, functions, data structures, algorithmic thinking, and how computers actually work. Taught in Python, but the concepts are universal.

Prerequisites: none — start here Capstone: a command-line expense tracker
Stage 2 · The Web & Tools
02Git Mastery
7 sections · Git & GitHub

Version control, the tool every developer uses every day. Snapshots, branching, collaboration, and rewriting history — up to a full professional team workflow.

Prerequisites: Foundations (helpful, not required) Capstone: run a real feature-branch workflow with PRs & conflict resolution
03HTML Mastery
11 sections · HTML

The structure of every web page. Semantic markup, forms, tables, media, and accessibility — the foundation everything visual on the web is built on.

Prerequisites: none Capstone: a multi-page semantic, accessible website
04CSS Mastery
13 sections · CSS

Making the web beautiful. The box model, flexbox, grid, responsive design, animations, and modern CSS — with 33 live, editable demos. Style the very site you built in HTML.

Prerequisites: HTML Mastery Capstone: three-era responsive layout project
Stage 3 · Programming for Real
05JavaScript Mastery
20 sections · JavaScript

The language of the web. From fundamentals through async, the DOM, modern JavaScript, and Node — everything you need to make websites interactive and build back-ends.

Prerequisites: Foundations, HTML, CSS Capstone: TaskFlow — a drag-and-drop Kanban app
06SQL Mastery
15 sections · SQL

How real applications store data. Queries, joins, design, indexes, transactions — the database skills every back-end developer needs.

Prerequisites: Foundations Capstone: design & query a real relational database

07System Design Mastery

How large systems are built. Scaling, databases, caching, queues, and eleven real case studies (URL shortener, chat, news feed, ride-sharing, and more). The senior-engineer skillset.

After you finish

You'll have eight courses' worth of knowledge and a portfolio of real, deployed projects. To turn that into a job:

  • Deploy every capstone. A live URL beats a description every time. Put them on GitHub with clean READMEs.
  • Pick a specialty. Frontend (React/TypeScript), back-end (APIs, databases), or AI engineering — go deeper in one direction.
  • Contribute to open source. One merged pull request to a real project is worth more than any certificate.
  • Build one bigger thing. Combine your skills into a single full-stack project you're proud of — that's your headline portfolio piece.
  • Practice interviewing. The System Design course prepared you for design interviews; practice coding problems alongside.

Free and open source, forever. If these courses helped you, consider supporting the project — entirely optional, only if you want to.
